# Astro-Dating — верификация исторических датировок через астрономию

Проект для независимой проверки хронологии древних событий путём сопоставления описаний в летописях (затмения, появления комет, сверхновые, положения планет и звёзд) с точными астрономическими расчётами по современным эфемеридам и каталогам.

## Первый результат: датировка каталога Альмагеста

Методом Морозова-Дамбиса (минимум RMS невязки звёздных координат при переборе эпох) получена дата составления каталога:

- **Быстрые звёзды (6 штук): −120 ± 100 CE** → эпоха Гиппарха (~−130)
- Опровергнута гипотеза Морозова о средневековом происхождении (+800)
- Подтверждена гипотеза Дамбиса-Ефремова: Птолемей скопировал каталог Гиппарха

См. [docs/REPORT.md](docs/REPORT.md) — полный научный отчёт.

## Запуск

```bash
pip install -r requirements.txt
python src/almagest_date.py
```

## Лицензия и воспроизводимость

Все данные открытые: Toomer 1984 (перевод Альмагеста), Hipparcos Catalog (ESA 1997).
Код воспроизводит результат за <1 секунды на любом ноутбуке.
