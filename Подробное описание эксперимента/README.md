# Подробное описание эксперимента

Материалы к работе **«Астрономическая верификация мировой хронологии: 16 якорных событий в диапазоне 5000 лет»** (Дмитриев А.А., ГПНТБ России, 16 апреля 2026 г.)

## Состав папки

| Файл / папка | Содержимое |
|---|---|
| **`Итоговый_документ.docx`** | **Word-документ для печати/публикации**: текст + 9 интегрированных графиков с подробными подписями, оглавление |
| `Итоговый_документ.md` | Markdown-исходник Word-документа (для редактирования и регенерации) |
| `Подробное_описание_эксперимента.md` | Исходная глава (~2100 слов, 6 разделов) без графиков |
| `Научный_доклад.md` | Доклад на 15–20 минут (15 слайдов) с описанием инфографики |
| `графики/` | 7 новых научных графиков (200 dpi) + 3 оригинальных + README с привязкой к слайдам |
| `результаты/` | MD-отчёты и JSON по каждому якорному событию |
| `исходный_код/` | Python-скрипты: 9 вычислительных модулей + `generate_graphs.py` (генератор графиков) |

## Регенерация Word-документа

```bash
cd "Подробное описание эксперимента"
pandoc "Итоговый_документ.md" -o "Итоговый_документ.docx" --toc --toc-depth=2 -V lang=ru
```
