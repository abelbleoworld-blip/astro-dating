Главный вывод

На выборке из 16 астрономически датируемых событий из 8 независимых цивилизаций в диапазоне от 3067 года до н.э. до 1986 года н.э. получено полное согласование с традиционной хронологической шкалой в пределах астрономической точности.

«Новая хронология» А.Т. Фоменко, требующая 1000-летнего сдвига античности, математически несовместима с физикой на каждом из проверенных якорей.

Peer-review

Две независимые системы

MacBook (macOS, astropy 6) и Beelink (Windows, astropy 7) параллельно считали с разными стеками. Совпадение исключает программные ошибки.

Воспроизводимость

Всё открыто. Все данные в публичном доступе. Весь код — Python. Время расчёта — менее 30 минут на любом ноутбуке. Любой школьник или студент может проверить сам. 
git clone https://github.com/abelbleoworld-blip/astro-dating.git
cd astro-dating
pip install -r requirements.txt
python src/almagest_date.py       # Датировка каталога Альмагеста
python src/halley_chronology.py   # 30 возвращений Галлея
python src/maya_dresden.py        # Майянская хронология
