# P1: Ласко — звёздный бык палеолита

**Дата:** 2026-04-19 05:01

## Гипотеза

Шесть точек над быком в Зале Быков = Плеяды.
Бык = Телец. Глаз быка = Альдебаран. V-точки = Гиады.
(Раппенглюк, 1996; д'Юи, 2016)

## Результаты

| Проверка | Результат |
|---|---|
| Видимость над Ласко (45°N) | ✓ обе звезды видны |
| Взаимное положение Плеяды-Альдебаран | ✓ 9.8° — неизменно |
| Собственное движение Альдебарана | 0.9° — узнаваем |
| Прецессия за 17 000 лет | 238° (0.66 оборота) |
| Датировка по прецессии | ⚠ невозможна (нет абсолютной ориентации) |

## Вывод

Гипотеза **астрономически непротиворечива**.
Человек верхнего палеолита мог наблюдать Плеяды и Альдебаран
в том же взаимном расположении, что и мы.
Если интерпретация верна — это древнейший астрономический
документ: **~17 000 лет**.
