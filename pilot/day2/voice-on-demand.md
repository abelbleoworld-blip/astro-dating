# 🎙 Voice — голосовые заметки ПО КОМАНДЕ
**Заменяет:** Whisper 24/7 (24/7-автозапись отключена пользователем 2026-04-17 03:47).
**Принцип:** записывает только когда **ты сам** запустил команду.

## Команды (3 штуки, всё)

```bash
voice            # запись пока не нажмёшь Enter — остановиться
voice 30         # запись ровно 30 секунд (для коротких заметок)
voice -q         # тихий режим: записать, сохранить, ничего не показывать
voice "идея"     # с тегом (по умолчанию тег ON-DEMAND)
v                # короткий алиас (тот же скрипт)
```

## Где сохраняется

`~/Documents/Projects/mesh-memory/voice/2026-04-17.txt`

Формат строки:
```
[2026-04-17_03-50-22] (ON-DEMAND) текст транскрипта здесь
[2026-04-17_04-15-08] (идея) другая запись с пользовательским тегом
```

## Workflow в реальной жизни

| Ситуация | Команда |
|---|---|
| Идея пока пьёшь кофе | `voice "идея"` → говоришь → Enter → сохранено |
| Долгий monolog (5-10 мин думаю вслух) | `voice` → говоришь → Enter |
| Быстрая мысль 10 сек | `voice 10` |
| Записать встречу 30 мин | `voice 1800` |
| Что-то сказал в фоне случайно — НЕ хочу сохранять | Ctrl+C во время записи |

## Поиск по записям

```bash
# Что я говорил про D1
grep -i "D1" ~/Documents/Projects/mesh-memory/voice/*.txt

# Все записи с тегом "идея"
grep "(идея)" ~/Documents/Projects/mesh-memory/voice/*.txt

# Сегодняшние записи
cat ~/Documents/Projects/mesh-memory/voice/$(date +%Y-%m-%d).txt
```

## Чем отличается от 24/7

| Аспект | 24/7 (отключено) | On-demand (текущий) |
|---|---|---|
| Когда пишет | Постоянно, 5-мин чанками | Только когда ты `voice` |
| Privacy | Слышит всё | Только то что ты сам решил записать |
| Шум/мусор | Много фоновой ерунды | Только осмысленное |
| Battery / CPU | Постоянная нагрузка | Только во время записи |
| Чужие голоса в комнате | Записывает | Не записывает (пока ты не нажал) |
| Контроль | Низкий | **Полный** |

## Если нужен hotkey (не команда в терминале)

Установить через **Apple Shortcuts.app** или **Karabiner-Elements**:
- Создать Shortcut → Run Shell Script: `/Users/macbookpro14/bin/voice 60`
- Назначить hotkey (например ⌘⌥V)
- Тогда не нужно идти в терминал, нажал hotkey → 60 сек записи → транскрипт в mesh

(Hotkey НЕ настроен по умолчанию — настроить если захочешь.)

## Откат / удаление

```bash
rm ~/bin/voice ~/bin/v   # удалить скрипты
# voice/ файлы остаются — удалять их вручную если не нужны
```

## Что делает скрипт

1. Старт sox запись с микрофона (моно 16 кГц — Whisper-native формат)
2. Ждёт твой Enter (или таймаут если задал длину)
3. Прогоняет через `whisper-cli` с моделью `ggml-base.bin`
4. Дописывает результат в `voice/yyyy-mm-dd.txt`
5. Удаляет временный wav-файл (только текст остаётся)
6. Показывает что записал (если не `-q`)
