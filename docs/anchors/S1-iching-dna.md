# S1 — Структурная конвергенция кодов: И-Цзин ↔ ДНК

**Класс якоря:** новый — *информационно-структурный* (не астрономический).
**Статус:** скелет, метод не реализован.
**Дата:** 2026-04-17

## Тезис

Две системы, разделённые ~4 млрд лет и не имеющие общего происхождения, дают одно и то же пространство кодов мощностью **64**:

| Система | Алфавит | База | Длина «слова» | Мощность |
|---|---|---|---|---|
| ДНК (генетический код) | A, T, G, C | 4 | 3 (триплет/кодон) | 4³ = **64** |
| И-Цзин (гадательный код) | Инь, Ян | 2 | 6 (гексаграмма) | 2⁶ = **64** |

Дополнительно: **отображение 64 → 20** (для ДНК — аминокислоты; для И-Цзина — традиционные классы гексаграмм по Шао Юну) даёт сравнимые редукции.

## Что является «якорем»

В отличие от A4/R1/C1 — здесь верифицируется не **датировка события**, а **структурный факт**:
> Совпадение мощности и редукционной структуры двух независимых дискретных кодов не является случайным с p < ε для произвольно подобранной модели «случайная пара кодирующих систем».

Якорь даёт **методологический аргумент**: дискретное кодирование — конвергентное явление, а не уникальное свойство европейской формальной науки. Это косвенно поддерживает доверие к точности нечисловых древних записей (летописи, канон эпонимов), на которых построены астрономические якоря A1..M3.

## Метод проверки (mesh)

Узел A (Mac) и узел B (Beelink) считают независимо:

1. **Биекция 64↔64.** Построить отображение кодонов на гексаграммы по двум независимым правилам:
   - правило Шёнбергера (1973): пурин/пиримидин → Инь/Ян, позиция в кодоне → позиция в триаде линий.
   - правило Стивенса (2016): ATGC → бинарные пары по водородным связям.
2. **Симметрии.** Проверить сохранение свойств:
   - комплементарность ДНК (A↔T, G↔C) → сохраняет ли инверсию Инь/Ян?
   - вырожденность кода (3-я позиция кодона) → отображается ли в специальный класс гексаграмм?
3. **Статистический тест.** Случайная пара (4-арный код длины 3, 2-арный длины 6) даёт 64=64 автоматически — но симметрии не обязаны совпадать. Посчитать долю случайных биекций, сохраняющих обе симметрии. Если < 1%, — конвергенция нетривиальна.
4. **Cross-check.** Узлы должны получить одинаковые числа. Расхождение — флаг ошибки реализации.

## Ограничения

- Биекция 64↔64 построима **многими способами** — выбор «канонической» — методологически нагруженный шаг. Без независимого критерия часть симметрий можно «настроить» под результат.
- Связь «культурный код древних → точность их астрономических наблюдений» — **гипотеза**, не доказательство. Якорь даёт гипотезу для обсуждения, не верификацию в смысле A1..M3.

## Связь с другими якорями

- **Параллель к C1 (Crab 1054).** C1 — кросс-культурная синхронизация события (5 культур независимо). S1 — кросс-кодовая конвергенция структуры. Разные плоскости, общий мотив: независимость наблюдателей.
- **Контраст к Фоменко.** Если структуры дискретного кодирования возникают конвергентно во многих культурах, тезис «вся древность сфальсифицирована единым центром» становится ещё менее правдоподобным.
