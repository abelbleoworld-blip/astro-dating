## 1. Тезис CIV-1 — кривая цивилизации через точность астрономии
Класс: мета-якорь (использует ВСЕ остальные якоря как data points). Дата: 2026-04-17.
Точность астрономических наблюдений = proxy уровня цивилизации. Измеримо.
Чем точнее культура знает небо — тем эффективнее предсказывает — тем она мощнее.
Небо = первый компьютер: время, навигация, календарь, предсказание, ритуал.
| Эпоха | Инструмент | Ёмкость (бит) | Точность |
|---|---|---|---|
| −3000 | Стоунхендж (50 камней) | ~6 бит | ~5° азимутов |
| −1000 | MUL.APIN (66 звёзд) | ~18 бит | ~2° |
| −150 | Антикитерский механизм (30+ шестерёнок) | ~10 бит предсказания | ~1° планет |
| −130 | Альмагест Гиппарх (1022 звезды) | ~20 Кбит | **RMS 1.23°** (наш расчёт) |
| +1000 | Дрезденский кодекс Майя | ~9 Кбит | ~0.1 дня (Венера) |
| +1437 | Улугбек (1018 звёзд) | ~20 Кбит | ~0.5° |
| +1602 | Тихо Браге (777 звёзд) | ~25 Кбит | ~1' (arcminute) |
| +1725 | Флэмстид (3310 звёзд) | ~50 Кбит | ~10" (arcsec) |
| +1997 | Hipparcos (118218 звёзд) | ~10 Мбит | ~1 mas |
| +2022 | Gaia DR3 (1.8 млрд) | ~100 Гбит | ~0.02 mas |
Рост: ~6 бит → ~100 Гбит за 5000 лет = удвоение каждые ~150 лет.
Это экспоненциальный закон. S-кривая или чистая экспонента — вопрос для анализа.
След древних в сегодня: Saros→NASA, 360°→GPS, 60мин→часы, 7дней→ISO 8601.
Мета-связка: CIV-1 использует A4, J1, M2, M3, C1, R1, Maya, China, D1 как points.
Каждый наш якорь = одна точка на кривой цивилизационного роста.
## 2. Метод (кривая + regression + предсказание)
Шаг 1. Собрать ~15-20 data points: (год, инструмент, ёмкость_бит, точность_угл_сек).
Ёмкость = log₂(число_объектов × параметров × бит_на_параметр). Точность = 1/угловая_ошибка.
Шаг 2. Fit экспоненциальной y = a × exp(b × year) и логистической y = L / (1 + exp(-k(x-x0))).
Шаг 3. Residuals: какие цивилизации ВЫШЕ тренда (скачки), какие ниже (тёмные века)?
Скачки ожидаемые: Гиппарх (−130), Тихо (1602), Hipparcos (1997), Gaia (2022).
Провалы ожидаемые: Dark Ages (500-900 CE), Монгольские нашествия (1220-1350).
Шаг 4. Cross-match с Grand Solar Minima из SOL-1: совпадают ли «тёмные века» с Spörer/Maunder?
Если да → солнечная активность → климат → цивилизационный спад → меньше астрономии.
Шаг 5. Экстраполяция: когда следующий скачок? Gaia DR4 (~2030)? Gravitational wave astronomy?
Шаг 6. Mesh peer-review: Mac строит regression, Beelink строит bootstrap confidence intervals.
Стек: Python + scipy.optimize.curve_fit + numpy + matplotlib.
Данные: все из наших якорей + Verbunt 2012 + литература по истории астрономии.
Время реализации: ~2-3 часа кода + 1 час анализа.
Стоимость: $0 (все данные уже в repo или в литературе).
Ограничение: «ёмкость» — наша оценка, не стандартная метрика. Нужно обосновать формулу.
Ограничение: выборка biased к культурам с письменными записями (нет Африки, Океании).
Ограничение: regression на 15-20 точках — шумный, широкие CI. Bootstrap обязателен.
Результат не «открытие» а «визуализация» + «framework для измерения». Но framework новый.
## 3. Литература и данные
Источники данных (по инструментам, уже в repo или легкодоступны):
1. Ruggles C. *Astronomy in Prehistoric Britain*. Yale 1999 (Стоунхендж).
2. Hunger H., Pingree D. *Astral Sciences in Mesopotamia*. Brill 1999 (MUL.APIN).
3. Freeth T. et al. *Decoding the Antikythera Mechanism*. Nature 2006.
4. Toomer G.J. *Ptolemy's Almagest*. Princeton 1984. + наш A4 расчёт (RMS 1.23°).
5. Verbunt F., van Gent R. *Star catalogues of Ptolemy and Ulugh Beg*. A&A 2012.
6. Aveni A.F. *Skywatchers of Ancient Mexico*. U.Texas 1980 (Дрезден).
7. Thoren V. *The Lord of Uraniborg: Tycho Brahe*. Cambridge 1990.
8. ESA. *Hipparcos Catalog*. SP-1200. 1997.
9. Gaia Collaboration. *Gaia DR3*. A&A 2023.
10. Shannon C.E. *Mathematical Theory of Communication*. Bell Sys. 1948 (формула бит).
Ограничения литературы:
- Нет единого каталога «информационной ёмкости» астрономических систем (мы первые?).
- Needham «Science and Civilisation in China» (1959) даёт качественные описания, не количественные.
- Hoskin «Cambridge Illustrated History of Astronomy» (1997) — обзор без метрик.
Данные в repo: A4 (1022 звезды Verbunt), Halley (30 появлений), sunspots (103 записи).
Данные нужно добавить: Стоунхендж (из Ruggles), Антикитерский (из Freeth), Тихо (из Thoren).
