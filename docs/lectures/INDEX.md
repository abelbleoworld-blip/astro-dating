## Как использовать серию
- **Новая Claude-сессия в этом проекте** — читать в порядке L1 → L8, занимает 30 минут
- **Объяснить кому-то проект за час** — L1 + L2 + L8 (методологический фундамент)
- **Применить принципы к новому проекту** — L8 (выбор шкалы) + L2 (если 2ⁿ)
- **Подготовить публикацию** — L6 + L7
- **Расширить mesh** — L4 + L5
## Принципиальная позиция автора (формализована в L1)
> «Нужно проверить без тени сомнения, нужно извне получить подтверждение — тогда то наука?»
Уровни L1-L4 применяются ко **всему** что выходит из проекта.
Утверждения от L1 (один прогон) до L4 (мульти-групп) маркируются явно: «согласуется», «воспроизводится локально», «принят peer-review» — никогда «доказано» без L3+.
## Распространение лекций (через mesh-memory)
После pilot v0.1 успеха — копия `lectures/` в `mesh-memory/knowledge/lectures/`.
Все Claude-сессии на всех машинах подгрузят серию автоматически (mesh-memory autopull 60s).
Это одна из практических реализаций L8 → L4 (mesh-распространение знания).
