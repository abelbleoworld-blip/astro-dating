# Информационное плато naked-eye астрономии

**Ключевой аналитический инсайт проекта astro-dating.**

## Тезис

Гиппарх (~−130 CE) создал каталог звёзд, который достиг **физического предела** информационной ёмкости naked-eye астрометрии. После него — **1600 лет плато** без качественного роста. Перелом — телескоп (1609).

## Три причины плато

### 1. Физический предел инструментов

Без телескопа угловое разрешение человеческого глаза + атмосфера + механическая точность армиллярной сферы/квадранта/секстанта упирались в **~1–2 arcmin**. Точность Гиппарха (~3600 arcsec ≈ 1°) была близка к этому пределу. Улучшать координаты на порядок без новой оптики — невозможно.

### 2. Отсутствие мотивации

Альмагест стал «библией» на 1400+ лет. Зачем новый каталог, если старый «работает»? Астрономия в основном обслуживала астрологию, календарь и навигацию. Никто не финансировал десятилетия наблюдений одной команды на уровне Тихо.

### 3. Технологические и социальные барьеры

- Нет точной механики для больших инструментов (до Ренессанса)
- Нет печатного станка → рукописные копии искажались
- Научный метод (систематическая ошибка, повторяемость, рецензирование) ещё не сформировался

## Перелом ~1600

Телескоп (1609–1610) + печатный станок + научная революция сломали **оба ограничения** одновременно:

1. **Оптика** → разрешение ушло в arcsec и лучше
2. **Печать + наука** → данные тиражируемы, проверяемы, накапливаются экспоненциально

## Связь с A4 (1022 звезды)

Наш результат (все 1022 звезды, RMS 1.23°, лучшая эпоха 50 CE) подтверждает: **все naked-eye каталоги стоят на одном плато**. Точность ~1° — это не ошибка наблюдателя, это предел технологии.

Именно поэтому метод Дамбиса работает: proper motions быстрых звёзд за 2000 лет (~1.3° для Арктура) **сопоставимы с точностью каталога** (~1°). Это «окно возможности» — и оно закрывается к эпохе телескопа (когда точность каталогов >> pm за тот же период).
