# L3 Roadmap — путь к независимой внешней верификации

**Цель:** перевести проект astro-dating с уровня L2 (mesh peer-review одного автора на двух машинах) на уровень **L3** — независимая третья сторона воспроизводит результаты из открытых данных и подтверждает либо опровергает.

**Дата:** 2026-04-17
**Принцип:** см. memory `feedback_external_verification.md`

## Иерархия независимости (напоминание)

| L | Что это | Статус для проекта |
|---|---|---|
| L1 | один автор, один прогон | — пройден |
| L2 | один автор, mesh (Mac + Beelink) | ✅ R1, проектируется для S1, D1 |
| **L3** | **третья сторона воспроизводит из открытых данных** | ⏳ цель этой дорожной карты |
| L4 | независимые группы разными методами | ✅ только C1 (Crab 1054 — 5 культур + Hubble) |

## Каналы L3 — параллельные дорожки

### Дорожка А — публикация в peer-review журналах

| Журнал | Профиль | Язык | Срок | Стоимость | Сила | Статус |
|---|---|---|---|---|---|---|
| **Journal for the History of Astronomy** (Sage, UK) | мировой профильный, Dambis & Efremov 2000 публиковались тут | англ. | 6-12 мес | open access ~$2k или free behind paywall | L3 высокий | англ. версия PAPER-v2 нужна |
| **Mediterranean Archaeology and Archaeometry** | археоастрономия, более доступный | англ. | 4-8 мес | низкая или 0 | L3 средний | проще пройти |
| **Archaeoastronomy** (Univ. of Texas) | спец. журнал | англ. | 6-12 мес | — | L3 средний | — |
| **ВАК-журналы РФ** | "Историко-астрономические исследования" (ИИЕТ РАН), "Земля и Вселенная" | рус. | 4-9 мес | 0 или ~30к₽ | L3 в РФ-сообществе | `ARTICLE-VAK.md` готов |

### Дорожка Б — препринт + цитируемость

| Канал | Описание | Срок | Стоимость | Заметки |
|---|---|---|---|---|
| **arXiv** (`physics.hist-ph` или `astro-ph.IM`) | глобальный препринт-сервер | 1-3 дня после endorsement | 0 | требует endorsement от автора, уже публиковавшегося в категории. Можно одновременно с письмом Дамбису |
| **ResearchGate** | соцсеть исследователей, индексируется | мгновенно | 0 | слабее arXiv, но обратная связь от профильных подписчиков |
| **Zenodo** | европейский архив с DOI | 1 день | 0 | даёт DOI на dataset/код, ссылается на GitHub |

### Дорожка В — прямые контакты с экспертами

| Эксперт | Где | Связь | Почему он | Сила сигнала если ответит |
|---|---|---|---|---|
| **А.К. Дамбис** | ГАИШ МГУ, Москва | через ГАИШ МГУ → отдел переменных звёзд | автор метода Дамбис-Ефремов (2000), на котором стоит якорь A4 | **L3.5** — отзыв родителя метода |
| **F.R. Stephenson** | Durham University (эмерит) | через сайт Durham | мировой авторитет по историческим затмениям, цитируется в reference list | L3.5 |
| **Alexander Jones** | NYU (ISAW) | через сайт NYU | Антикитерский механизм, древняя астрономия | L3 для D1 |
| **R.R. Newton (наследие)** | — | — | "The Crime of Claudius Ptolemy" 1977 — критическая школа, к которой проект частично примыкает | — нет живого носителя |

### Дорожка Г — независимая воспроизводимость кода

| Шаг | Что | Что подтверждает |
|---|---|---|
| Г1 | Переписать ключевые скрипты под **JPL Horizons** web API (REST) вместо локального skyfield+DE422 | независимость от выбора эфемерид |
| Г2 | Параллельная реализация на **Stellarium-scripts** или **PyEphem** | независимость от стека библиотек |
| Г3 | Открытый Jupyter notebook на Google Colab — любой может запустить из браузера | устранение барьера установки |
| Г4 | Hash-ссылки на конкретные версии Hipparcos / NASA Eclipse Catalog в `data/SOURCES.md` | воспроизводимость данных |

Это всё ещё один автор → **L2.7**, но необходимое условие для L3 (без этого внешний рецензент не сможет воспроизвести).

### Дорожка Д — AI-кросс-рецензия (псевдо-L3, дешёвая)

Промпт во все три модели одновременно:
> "Прочитай PAPER-v2.md проекта astro-dating. Найди: (1) методологические дыры; (2) статистические ошибки; (3) пропущенную релевантную литературу; (4) места, где аргумент против Фоменко слабее, чем заявлено. Рецензия в стиле peer-reviewer JHA."

| Модель | Стоимость | Что ловит |
|---|---|---|
| **Grok-4** (xAI) | $5-15 | другая база обучения, может найти западные источники |
| **Gemini-2.5 Pro** | $5-15 | сильна в физике/астрономии |
| **GPT-5** (OpenAI) | $10-30 | другой подход к рецензированию |
| **DeepSeek-R3** | $1-3 | дешёво, китайский корпус — может найти источники по Сун-эпохе |

⚠ Это **не L3** — модели обучены на близких корпусах, формальной независимости нет. Но для **sanity check** перед отправкой настоящему рецензенту — лучшее соотношение цена/польза.

## Критерии успеха L3

| Сигнал | Уровень |
|---|---|
| arXiv preprint опубликован, цитируется хотя бы 1 раз | L2.8 |
| Дамбис/Stephenson ответили письмом с содержательным комментарием | **L3.0** |
| Принят в Mediterranean Archaeology and Archaeometry или ВАК | **L3.2** |
| Принят в JHA после peer-review | **L3.5** |
| Независимая группа повторила результат и опубликовала | **L4.0** |

## Что НЕ считается L3

- Лайки в соцсетях, репосты в Telegram/Twitter — L0
- Положительный отзыв LLM (даже трёх разных) — L2.5 псевдо
- Положительный отзыв нерофессионала ("прочитал, мне нравится") — L1.5
- Цитирование в популярной книге без проверки — L1.5
- Положительный отзыв коллеги по проекту/мессенджер-чату — L2
