# SOL-1 Pipeline — 5 этапов по источникам данных
**Принцип:** каждый этап = один источник, один скрипт, один результат. Поэтапно.
**Дата:** 2026-04-17. Статус: план.

## 5 этапов

| Этап | Источник | Записей | Период | Данные | Срок |
|---|---|---|---|---|---|
| **E1** | Солнечные пятна (bare-eye) | 112 | 28 BCE — 1610 CE | Yau & Stephenson 1988 | **неделя 1** |
| **E2** | Аврора (северное сияние) | ~2000 | 567 CE — 1900 CE | Silverman 1992, Keimatsu 1970-76 | неделя 2 |
| **E3** | Затмения с описанием короны | ~200 | 700 BCE — 1900 CE | Stephenson 1997, Espenak NASA | неделя 3 |

## Каждый этап = 4 шага

```
[Скачать данные] → [Cross-match с якорями astro-dating] → [FFT/wavelet] → [Отчёт]
```

1. `src/sol1_eN_download.py` — скачать/распарсить источник → CSV в `data/`
2. `src/sol1_eN_verify.py` — cross-match дат с 11+ якорями (±10 лет) → маркировка
3. `src/sol1_eN_fft.py` — FFT + wavelet верифицированных записей → периоды
4. `results/sol1_eN_report.md` — отчёт + графики → что нашли

## E1 Солнечные пятна (первый, самый быстрый)

**Данные:** Yau K.K.C., Stephenson F.R. 1988 «A Revised Catalogue of Far Eastern Sunspot Records» (QJRAS 29, 175-197). Также: Vaquero & Vázquez 2009 (Springer).

**Формат записи:** год, месяц (если известен), источник (Хань шу, Цзинь шу, Сун ши и т.д.), описание.

**Cross-match:** для каждой записи проверяем — есть ли рядом (±10 лет) якорь astro-dating?
- J1 Бур-Сагале 763 BCE (ассирийский канон фиксирует шкалу)
- C1 Crab 1054 (сверхновая, те же хроники)
- China chronology (MUL.APIN аналоги)
- Halley apparitions (30 кометных якорей)
- A4 Альмагест (звёздный каталог = точка на оси)

**Ожидаемый результат:** ~30-50 записей пятен в пределах ±10 лет от якоря → verified.

**FFT целевые частоты:** 11 лет (Швабе), 22 (Хэйл), 88 (Глейсберг), 210 (Сюсс).

## E2 Аврора (второй, побольше)

**Данные:** Silverman S.M. 1992 «Secular variation of the aurora» (Rev. Geophys.), Keimatsu M. 1970-76 каталог дальневосточных аврор. Также: Fritz H. 1873.

**Особенность:** аврора = прокси геомагнитной активности → прямо пропорциональна солнечной. Но: низкие широты = сильные бури (только максимумы).

**Cross-match:** те же якоря + европейские летописи (Gregory of Tours 580s).

## E3 Затмения + корона (третий)

**Данные:** Stephenson 1997 «Historical Eclipses and Earth's Rotation». Espenak Five Millennium Catalog.

**Особенность уникальная:** при затмении на МИНИМУМЕ солнечного цикла корона маленькая (круглая). На МАКСИМУМЕ — лучистая, обширная. Описания короны в хрониках = **прямое наблюдение фазы цикла**.

**Cross-match:** затмения УЖЕ являются якорями (J1, A1, R1, M2) → верификация автоматическая.

## E4 Carbon-14 (четвёртый, физический прокси)

**Данные:** IntCal20 (Reimer et al. 2020, Radiocarbon). Скачать: https://www.intcal.org/

**Особенность:** C14 в атмосфере = обратно пропорционален солнечной активности (больше активность → больше магнитное поле → меньше космических лучей → меньше C14 рождается). Непрерывная шкала 12000 лет с разрешением ~10 лет.

**Cross-match:** не с якорями (шкала в абсолютных годах через дендрохронологию), а **совпадение периодов** найденных на E1-E3 с C14-периодами.

## E5 Beryllium-10 (пятый, самый глубокий)

**Данные:** GISP2 ice core (Greenland), GRIP, EPICA Dome C (Antarctica). Finkel & Nishiizumi 1997 (JGR).

**Особенность:** Be10 = второй независимый прокси (те же космические лучи, другой химический путь). Если C14 и Be10 совпадают → L4 cross-validation.

**Глубина:** до 50000 лет. Но разрешение хуже (~50-100 лет).

## Финальный шаг: PREDICTION

После E1-E5 → суперпозиция всех источников:

```
src/sol1_prediction.py:
  1. Verified sunspots (E1) → периоды
  2. Verified aurora (E2) → периоды
  3. Eclipse corona (E3) → фаза цикла
  4. C14 (E4) → долгосрочные периоды
  5. Be10 (E5) → cross-check

  → wavelet decomposition → 
  → Schwabe + Hale + Gleissberg + Suess + Hallstatt суперпозиция →
  → экстраполяция → 
  → ПРОГНОЗ: следующий Grand Solar Minimum = год ± погрешность
```
