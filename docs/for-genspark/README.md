# Материалы для Genspark AI Slides — Astro-Dating

Пакет для генерации презентации через **genspark.ai → AI Slides** (работает через браузер, API закрыт Cloudflare).

## Что есть

| Файл | Для чего |
|---|---|
| **1-PROMPT.md** | Основной промпт — скопируй в форму создания презы |
| **2-FULL-BRIEF.md** | Полный научный контент по всем 16 якорям — приложи как Source |
| **3-SOURCES.md** | Библиография и ссылки на первоисточники — приложи как Source |
| **4-STYLE-REFERENCE.html** | Образец стиля слайда (1280×720, deep-blue + sky-blue) |

## Рабочий процесс

1. Открой https://www.genspark.ai/
2. Войди в аккаунт (нужен залогин, т.к. Genspark за CF)
3. Выбери **AI Slides** или **Super Agent**
4. Нажми «Загрузить источники» (Upload) → приложи файлы 2, 3, 4
5. В поле темы/промпта вставь содержимое файла **1-PROMPT.md**
6. Выбери модель — **GPT-5 Pro** или **Claude Opus 4.6** (если доступны)
7. Запусти генерацию → получи PPTX/PDF

## Ключевые параметры для Genspark

- **Язык**: русский
- **Количество слайдов**: 18-20
- **Стиль**: тёмный deep-blue (`#050b14`) + акцент sky-blue (`#4da8da`), шрифты Montserrat + Open Sans, Font Awesome для иконок
- **Формат**: 16:9 (1280×720)
- **Без градиентов**, только плоские цвета с opacity

## Если что-то недогенерилось

Подай дополнительный промпт:
> Добавь слайды 10-15 по индивидуальным якорям (Альмагест, Дендера, комета Галлея, сверхновая 1054, затмение 1185, Махабхарата, Рамаяна) с формулами и таблицами

Или попроси перегенерить конкретный слайд:
> Слайд 5 (Бур-Сагале): добавь конкретные числа — 15 июня 763 BCE, разделение 0.028°, время 13:45 UTC, координаты Ниневии 36.36° с.ш. 43.15° в.д.

## Запасной путь — наш AbelForge

Если Genspark работает плохо / долго / платно — материалы те же можно скормить нашему локальному генератору:

```bash
# из ~/Documents/Projects/AbelForge/
node src/cli.js --topic "Астрономическая верификация мировой хронологии: 16 якорных событий в диапазоне 5000 лет" --context "$(cat ~/Documents/Projects/astro-dating/docs/for-genspark/2-FULL-BRIEF.md)" --slides 20 --tier max --template tech
```

(После того как допилим HTML-пайплайн в AbelForge — будет сразу HTML а не PPTX.)
